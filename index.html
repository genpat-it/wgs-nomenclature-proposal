From a distance matrix to a path-code

Hierarchical clustering of the distances, then a cut at each threshold (coarse → fine). Each cut adds one digit. The name A.B.C is the path down the tree: shared prefix = shared ancestry. Move the sliders and watch the cut lines and the codes change.

Computed LIN codes

Each position is a fixed distance threshold. New samples only ever append deeper digits — they never renumber the prefixes above them. That is invariance "by construction" (see tab 3).

Distance distribution + thresholds

Each threshold should fall in a “gap” (few or no distances), not inside a pile of bars. Threshold in a gap = clean cut between real groups; threshold inside the bars = forcing (splits a real cluster).

“Many samples on the same level” — split or represent?

The real question is not which of the N samples, but are they above or below the threshold? Above the threshold → genuinely distinct → split (each gets its own digit). Below → one true cluster → pick the medoid (member with smallest summed distance), with a deterministic tie-break.

Single-linkage tree (cut at t)

R → one node per cluster at threshold t → its isolates. A red weight on an arc = a real split at that distance (≥2 clusters = SPLIT); a grey ≤t = no split, one cluster → pick the ★ medoid.

Invariance — frozen backbone vs recompute, and the blind spots

Same data, same new isolates — switch the mode and watch the difference. Frozen: new isolates are classified against fixed references → old codes never change. Recompute: the whole tree is rebuilt each time → re-clustering can merge clusters (the bridge) and rewrite codes. (Digit order uses the oldest member, not the medoid, so a medoid shift alone no longer renumbers siblings.) The last arrivals show the blind spots: far = a divergent lineage is seen (new top digit); nov = a novel group within t1 is framed under an existing lineage; twin = sub-threshold novelty is merged (resolution cap).

Segmented genomes — constellation & readable reassortment

For a segmented virus (BTV: 10 segments) each segment gets its own path-code. The name is the constellation of all segment codes. When one segment's lineage disagrees with the rest of the genome, that digit lights up — reassortment becomes readable in the name itself.

Per-segment trees (the reassortant sits on a different branch)

Each cell = that segment's lineage digit for that isolate. A cell flagged like this is in a different lineage from the rest of that isolate's genome → the segment reassorted in. The whole-genome name (constellation) carries the signal — no separate tree to consult.
